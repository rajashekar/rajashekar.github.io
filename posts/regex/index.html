    Below is overview on Regex standards

    Below are few commands and examples of Regular expressions. 1. Dot - . any character except newline

    1. \w \d \s - word, digit, whitespace
    Example:
substitution 
a. %s/\(\w\+\)/"\1"/g

before:
test test test

after:
"test" "test" "test"

b. %s/\(\d\+\)/"\1"/g
before:
test test test 
123 123 1234 123456

after:
test test test 
"123" "123" "1234" "123456"

    1. \W \D \S not word, digit, whitespace

    2. [abc] any of a, b, or c

    3. [^abc] not a, b, or c

    4. [a-g] character between a & g

    5. ^abc$ start / end of the string

    6. \t \n \r tab, linefeed, carriage return

    7. a* 0 or more

    8. a+ 1 or more

    9. a? 0 or 1

    10. a{5} exactly 5

    11. a{2,} 2 or more

    12. a{3,5} - matches 3, 4, 5

    13. & Back reference without grouping 

    :%s/test/"&"/g

before:
test | test | test | test | test | test 

after:
"test" | "test" | "test" | "test" | "test" | "test"
    1. Grouping - (abc) \1 backreference
    Example - 
^/sams/cart/(cart.jsp)/test/\1$

matches - 
/sams/cart/cart.jsp/test/cart.jsp

Not matches - 
/sams/cart/test/cart.jsp
/sams/cart/cart.jsp
/sams/cart/cart.jsp/test
    1. non-capturing group - (?:abc) 
    Example - 
^/sams/cart/(?:cart.jsp)/(test)/\1$

matches - 
/sams/cart/cart.jsp/test/test

not matches - 
/sams/cart/test/cart.jsp
/sams/cart/cart.jsp
/sams/cart/cart.jsp/test
    1. Negative lookahead - (?!abc) Example - 
    ^/sams/cart/(?!cart.jsp).*jsp$

matches - 
/sams/cart/test/cart.jsp
/sams/cart/cartService.jsp
/sams/cart/saveForLater.jsp

Not matches - 
/sams/cart/cart.jsp?xid=123&redirectURL=test.jsp
    1. Postive lookahead - (?=abc) Example - 
    ^/sams/cart/(?=cart.jsp).*jsp$

matches - 
/sams/cart/cart.jsp?xid=123&redirectURL=test.jsp

not matches - 
/sams/cart/test/cart.jsp
/sams/cart/cartService.jsp
/sams/cart/saveForLater.jsp
    1. Named Capturing Group - (?P<abc>.*) or {?<abc>.*) 
    Example: 

([^|]+\|){3}(?P<xyz>[^|]+)

test1 | test2 | test3 | test4 | test5 | test6 

xyz is test4
